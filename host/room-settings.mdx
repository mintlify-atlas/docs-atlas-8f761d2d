---
title: "Room settings"
description: "Configure your Squiggle room before the game starts."
---

As the host, you can adjust how the game plays before starting. All settings are in the **Host Settings** panel visible in the lobby. Changes take effect immediately and are synced to all players in the room.

<Warning>
Settings can only be changed while the game is in the **lobby** phase. Once you click **Start Game**, the settings are locked for that session.
</Warning>

## Settings reference

### Rounds

**Default: 3** — Options: 1–10

The number of full rounds to play. One round = every player draws once. A 3-round game with 4 players means 12 total drawing turns.

### Draw time

**Default: 80s** — Options: 30s, 45s, 60s, 80s, 100s, 120s

How long the active drawer has to draw before the round automatically ends. This timer is visible to all players as a countdown.

### Word choices

**Default: 3** — Options: 2–5

How many word options the drawer sees before each turn. A higher number gives the drawer more flexibility; a lower number creates more urgency.

### Max players

**Default: 8** — Options: 2–80

The maximum number of players allowed in the room at the same time. Players who try to join beyond this limit see a "Room is full" error.

### Hints

**Default: 2** — Options: 0–5

The number of letter hints revealed during a drawing turn. Set to **0** to disable hints entirely. When hints are set to 0, the hint interval setting is hidden.

### Hint interval

**Default: 20s** — Options: 10s, 15s, 20s, 25s, 30s

How many seconds pass between each hint reveal. Only visible when **Hints** is set to 1 or more.

**Example with 2 hints and a 20s interval:**
- At 20s — first letter revealed
- At 40s — second letter revealed

### Categories

**Default: Animals, Food, Objects** — Available: Animals, Food, Movies, Tech, Countries, Objects

Toggle which word categories are included in the drawing pool. At least one category must remain selected. Custom words (see below) are mixed in alongside whichever categories are active.

### Custom words

**Default: empty**

A comma-separated list of words to add to the pool. See [Custom words](/host/custom-words) for full details.
