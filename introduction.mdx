# Squiggle

**draw. guess. repeat.**

Squiggle is a minimal, open-source multiplayer draw & guess game inspired by skribbl.io. Players create private rooms, share a 6-character code, and take turns drawing while everyone else races to guess the word.

Built with Next.js 15, Firebase Firestore, and TypeScript — Squiggle is lightweight, real-time, and cheat-resistant by design.

## What is Squiggle?

Squiggle is a browser-based party game for 2–8 players. Each round, one player draws a secret word on a shared canvas while the others type guesses into the chat. Points are awarded based on how quickly you guess correctly — and the drawer earns points too, every time someone guesses right.

Games are organized into rooms. Every room has a unique 6-character code you share with friends. The host controls all game settings before the game starts.
